# GitGrid

递归扫描当前目录下所有 Git 子仓库，在一个终端表格里集中展示它们的分支、远端同步状态、本地改动和储藏。

## 特性

- 递归发现子仓库，自动忽略 `thirdparty` / `node_modules` / `build` 等无关目录
- 刷新时自动 `git fetch`（并行 + 超时），能真实检测到**远程有更新**的仓库
- 有远程更新（behind > 0）的仓库整行标红高亮，一眼定位
- 后台线程执行，UI 不卡顿

## 安装

```bash
pip install .
# 或开发模式（改代码立即生效）
pip install -e .
```

## 使用

在包含多个 Git 仓库的父目录下运行：

```bash
gitgrid
```

### 快捷键

| 键 | 作用 |
|----|------|
| `r` | 刷新状态（会执行 fetch） |
| `f` | 开关自动 fetch（关闭后只看本地缓存，速度快） |
| `q` | 退出 |
